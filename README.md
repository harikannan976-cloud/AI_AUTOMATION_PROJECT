# AI Automation Project

> **This entire project was created through scriptless AI automation using Claude and an MCP server — not a single line of code was written manually. Everything from project scaffolding and test authoring to configuration, documentation, git commits, and pushing to this repository was driven entirely by AI automation, with no human intervention in the code.**

This is a Playwright project for automation testing.

## What it includes

- End-to-end browser automation using Playwright
- Structured test architecture for account creation and login flows
- Headed and headless execution support
- Popup and overlay handling for real-world sites
- HTML report generation
- AI-assisted workflow support

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Install Playwright browsers:
   ```bash
   npx playwright install
   ```

3. Run tests:
   ```bash
   npm test
   ```

4. Run the Steve Madden account flow in headed mode:
   ```bash
   npx playwright test tests/stevemadden-account.spec.js --headed
   ```

5. Show report:
   ```bash
   npx playwright show-report
   ```

## How It Was Built

This project demonstrates end-to-end AI-driven development:

- **No manual coding** — every file was generated by Claude (Anthropic) via natural language prompts
- **MCP server integration** — Claude used an MCP server to orchestrate tool calls, file writes, and git operations
- **Full lifecycle automation** — project setup, test writing, config, documentation, commits, and push to GitHub were all handled by AI
- **Zero human code intervention** — from `npm init` to `git push`, the entire workflow ran through Claude Code automation
